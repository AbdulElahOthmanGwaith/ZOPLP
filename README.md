# ZOPLP - Social Media Application

ZOPLP is a modern social media application similar to Facebook, designed with unique features and a user-friendly interface.

## Features

- **Unlimited Friends**: Unlike traditional social media platforms, ZOPLP allows unlimited friend connections with no 5,000-friend limit.
- **Voice Comments**: Share your thoughts with voice comments in addition to text-based comments.
- **Arabic Language Support**: Full RTL (Right-to-Left) support for Arabic language users.
- **Modern Design**: Clean and intuitive interface with a blue and purple color scheme.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Language**: Arabic with RTL support
- **Design**: Responsive and modern UI

## Features Implemented

- User authentication and login system
- News feed with post creation
- Like, comment, and share functionality
- Friend counter and profile management
- Sidebar navigation
- Real-time interactions

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AbdulElahOthmanGwaith/ZOPLP.git
   cd ZOPLP
   ```

2. Open `index.html` in your web browser to run the application locally.
